# Pet Adoption
A simple and attractive pet adoption landing page built using HTML and CSS.

## ​Features
* **​Pet Profiles:** Displays detailed information for different pets like Jerry and Tom.
* **​Flexbox Layout:** Organized pet cards side-by-side using display: flex.
* **​Modern Design:** Includes rounded borders (border-radius) and a clean look.
* **​Custom Styling:** Highlighting key text with specific brand colors.
